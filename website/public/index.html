Help

Usage with monero-wallet

  1. Download the wallet Monero GUI/CLI
  2. In the wallet, go to Settings - Node - "Remote Node" - add remote node.
  3. Enter the following address as the remote node: node.monero.mbaraa.com.
  4. Use this port: 443.
  5. Go back to Node and click on the address.

Frequently Asked Questions

Who needs this node?

Anyone who wants to make a quick Monero transaction without needing to sync the entire blockchain on your device.

Is this safe? Can I trust you?

While using your own node is always the safest option, however, as you know running a Monero node isn't cheap, so if I wanted to run a scam I would've picked something cheaper to run and maintain, I initially did it when I was moving around without my main computer in hand and then I was like “Hmm what if someone else can also use this in cases like mine’.
